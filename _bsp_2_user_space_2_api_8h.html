Detailed Description

This BSP is a null/empty BSP to be used when compiling applications that are dependent on "Bsp/Api.h", by the compile target is a desktop OS running in user space.

DO NOT include this file directly! Instead include the generic BSP interface - src/Bsp/Api.h - and then configure your project's 'colony_map.h' to include THIS file.
